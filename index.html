Besoins du projet

Pour ce projet, le besoin était de créer un espace où les employés d'une même entreprise peuvent créer et partager du contenu sur des sujets qui les intéressent. J'ai choisi d'intégrer en plus une messagerie instantanée.

Fonctionnalités majeures

  • API REST sécurisée qui permet de créer, lire, modifier et supprimer du contenu
  • base de données SQL
  • gestion de comptes d'utilisateurs
  • gestion de sessions utilisateurs grâce aux jsonwebtokens
  • Single Wep Application
  • messagerie instantanée utilisant les WebSockets
